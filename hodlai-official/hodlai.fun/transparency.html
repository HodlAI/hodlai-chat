常见问题

为什么有两种验证方式？

Stripe 用于支付 AI API 服务商；BSCScan 用于链上回购销毁和营销推广。两种方式互补。

什么时候会回购销毁代币？

当 API 资金池充足且有盈余时，我们会根据市场情况动态执行回购销毁。

团队如何盈利？

团队不从交易税中提取费用。我们持有代币，项目增值即是我们的收益，确保利益一致。
