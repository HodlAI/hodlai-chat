常见问题

HodlAI API 稳定吗？上游是什么？

✅ 稳定可靠！当前主要使用 OpenAI、OpenRouter 官方渠道作为上游，全部通过官方渠道调用。所有计费与官方一致，如发现漏掉或不一致的情况，请联系我们修正。

签名安全吗？

✅ 完全安全！签名仅验证钱包所有权，不产生链上交易，不消耗 Gas，代币始终在你钱包中。

3% 交易税怎么用？

优先保障 AI API 资金池，持有者按持有量比例每日分享免费额度。盈余用于回购销毁或营销推广，团队 0 抽成。

额度怎么计算和释放？

基础额度：每$10持仓=$1×当前价格比例（币价≤$0.001时100%，≥$0.02时30%）。释放规则：0-5分钟0%→5分钟后10%→每小时+4%→24小时100%💎。买入不重置，卖出/转出才重置，曾卖出过最高80%。

额度用完怎么办？

① 等待每日 00:00 (北京时间) 重置 ② 增持代币获得更多额度 ③ 成为钻石手享受 100% 额度

我的对话数据安全吗？

🔒 绝对隐私！我们承诺不会记录、存储或分析任何对话数据。所有请求直接转发至上游 API，我们的服务器不做任何记录，你的隐私由我们全力守护。

什么是动态价格比例？

为保护早期持有者，币价越低额度比例越高。币价按$0.001档位计算：≤$0.001=100%，$0.005=82%，$0.01=63%，≥$0.02=30%。只有跨越档位时才重新计算，同档位内波动不影响现有额度。
