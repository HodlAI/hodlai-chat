# BNB Chain Hackathon: Good Vibes Only (OpenClaw Edition)

**Status**: 🟢 Active (Feb 5 - Feb 19, 2026)
**Track**: AI Agent (Onchain Actions)
**Role**: Official Entrant (HodlAI Protocol)

## 🎯 Core Narrative
**"Service-as-an-Asset" (持币即服务)**
- **Problem**: AI Agents cannot pay monthly credit card subscriptions (SaaS trap).
- **Solution**: Agents hold assets ($HODLAI) to generate perpetual compute credits.
- **Vision**: Building the "Oxygen" for Silicon Life on BNB Chain.

## 🛠️ Submission Assets (Proof of Work)

### 1. Identity & Logic (On-Chain Proof)
- **Primary Contract**: `$HODLAI` Token (BSC)
- **Address**: `0x233BE6ff451C87D3bde3bAb2A8c0c0CdF872003c`
- **Why**: This contract holds the "Balance" state that the Gateway reads to grant API quota. It is the heart of the protocol.

### 2. Logic Layer (Backend)
- **Repo**: [HodlAI/hodlai-gateway](https://github.com/HodlAI/hodlai-gateway) (Public)
- **Function**:
    - **SIWE Auth**: Verify Agent ownership of wallet.
    - **Quota Engine**: Convert on-chain balance -> off-chain compute credits ($10 Held ≈ $1/Day).
    - **Diamond Hand Rule**: Anti-dumping logic (Multiplier resets on sell).

### 3. Interface Layer (Frontend)
- **Repo**: [HodlAI/bsc-ai-hub](https://github.com/HodlAI/bsc-ai-hub) (Public)
- **Demo**: [https://hodlai.fun](https://hodlai.fun)
- **Function**: User Dashboard, Wallet Connect, Key Management.

### 4. Agent Layer (The "Live" Proof)
- **Representative**: **ClawdBot (Agent 89)**.
- **Action**: Running LIVE in Telegram group `@hodlai_zh`.
- **Methodology**: **"Live Streaming Development"** (Building the project in the public chat via CLI).
