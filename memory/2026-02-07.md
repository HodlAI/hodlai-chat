## 🛠️ Dev Log: hodlai-chat
- **Critical UX Fixes**:
  1.  **Session Persistence**: Fixed issue where refreshing the page loaded the "Home/Welcome" screen instead of the last active conversation.
      - *Root Cause*: `chatHistory` in Zustand store was initialized as `[]` effectively decoupling it from the persisted `sessions` list until a user interaction occurred.
      - *Fix*: Hydrated `chatHistory` directly from `sessions[0].messages` during store initialization.
  2.  **Auto-Scroll**: Fixed issue where page loaded at the top of history.
      - *Fix*: Implemented aggressive scroll-to-bottom logic using `requestAnimationFrame` + dual timeouts (100ms, 500ms) to handle rendering delays.
  3.  **Visual Alignment**: Fixed "Avatar vs Text" vertical misalignment.
      - *Fix*: Added `pt-[6px]` to Markdown container to align the first line of text with the avatar's visual center.
